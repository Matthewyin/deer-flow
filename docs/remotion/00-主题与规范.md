# DeerFlow 视频系列 — 主题与规范

> 所有视频共用此规范。每个视频的详细分镜见 `01-整体架构.md`、`02-带宽评估.md`、`03-运维知识库.md`、`04-远程探测.md`。

---

## 1. 视频参数

| 参数 | 值 | 原因 |
|------|-----|------|
| 分辨率 | 1920 × 1080 | YouTube/B站标准 |
| 帧率 | 30 fps | 技术动画不需要 60fps，渲染更快 |
| 时长 | 每个 40 秒（1200 frames） | 短视频节奏，信息密度高 |
| 输出格式 | H.264 MP4 | 通用兼容 |

## 2. 视觉风格

### 2.1 背景

```
渐变方向：从上到下
顶部颜色：#0a0e17（极深蓝黑）
底部颜色：#111827（深灰蓝）
叠加：微弱的网格线（opacity 0.05，间距 60px），营造技术图纸感
```

### 2.2 配色方案

每个组件类别对应一个固定颜色，所有视频统一：

| 类别 | 色值 | 用途 |
|------|------|------|
| 用户层 | `#60A5FA` (蓝) | 用户输入、浏览器 |
| Agent 层 | `#A78BFA` (紫) | Lead Agent、Dedi Agent、Middleware |
| MCP 工具层 | `#34D399` (绿) | MCP Server、Tools |
| 数据层 | `#FBBF24` (金黄) | ChromaDB、SQLite、MySQL |
| 基础设施 | `#F87171` (红) | Docker、Nginx、沙箱 |
| 外部服务 | `#FB923C` (橙) | Ollama、HuggingFace |
| 数据流粒子 | `#38BDF8` (亮蓝) | 连线上流动的粒子 |
| 文字-主 | `#F9FAFB` (近白) | 标题、正文 |
| 文字-次 | `#9CA3AF` (灰) | 注释、标签 |
| 代码/技术名 | `#34D399` (绿) | 类似终端绿色 |

### 2.3 字体

| 用途 | 字体 | 回退 |
|------|------|------|
| 中文正文/标题 | `Noto Sans SC` (Google Fonts) | system-ui |
| 英文/代码/技术名 | `JetBrains Mono` | monospace |

### 2.4 发光效果

所有架构节点卡片使用统一发光边框：

```css
box-shadow: 0 0 20px ${color}40,  /* 外发光，颜色 25% 透明度 */
            inset 0 0 20px ${color}15; /* 内发光，颜色 8% 透明度 */
border: 1px solid ${color}80;       /* 边框，颜色 50% 透明度 */
border-radius: 12px;
background: ${color}10;             /* 背景填充，颜色 6% 透明度 */
```

## 3. 动画参数

### 3.1 统一 timing 常量

```typescript
// 所有视频共用的动画参数
export const TIMING = {
  // 出现动画
  nodeSpringIn: { damping: 200, mass: 0.8, stiffness: 200 },
  nodeStaggerDelay: 3,   // 节点间延迟帧数（快速节奏）

  // 数据流
  flowDuration: 1.5,     // 粒子从 A 到 B 的秒数
  flowParticleSize: 4,    // 粒子半径 px

  // 转场
  transitionDuration: 10, // 转场帧数（快节奏）
  fadeInDuration: 8,      // 淡入帧数

  // 文字
  typewriterSpeed: 1,     // 每帧出现的字符数

  // 柱状图
  barStaggerDelay: 3,     // 柱子间延迟帧数
  barSpringConfig: { damping: 200, mass: 0.5 },
} as const;
```

### 3.2 禁止项

- ❌ CSS transitions / CSS animations / Tailwind 动画类名（Remotion 不支持）
- ❌ 第三方图表库自带动画（必须用 `useCurrentFrame()` 驱动）
- ❌ 超过 15 帧的转场（40 秒视频经不起浪费）

## 4. 共用组件清单

以下组件在 `src/components/` 中实现，所有视频复用：

### 4.1 ArchitectureNode

```
输入 props:
  - label: string          // 节点标题（如 "LangGraph Server"）
  - subtitle?: string      // 副标题（如 "Port 2024"）
  - color: string          // 主题色（来自 §2.2 配色）
  - icon?: ReactNode       // 可选图标
  - x: number              // 绝对定位 X
  - y: number              // 绝对定位 Y
  - width: number          // 卡片宽度
  - height: number         // 卡片高度

行为：
  - 整个卡片从 opacity 0 + scale(0.8) → opacity 1 + scale(1.0)
  - 使用 spring 动画（damping: 200）
  - 出现后边框持续微弱呼吸发光（周期 120 帧，opacity 在 0.3-0.6 间波动）
```

### 4.2 DataFlowLine

```
输入 props:
  - from: {x, y}          // 起点
  - to: {x, y}            // 终点
  - color?: string         // 默认 #38BDF8
  - curved?: boolean       // 是否贝塞尔曲线（默认 true）
  - curveOffset?: number   // 曲线偏移量（默认 80）
  - animated?: boolean     // 是否显示流动粒子（默认 true）
  - label?: string         // 连线上的标签文字
  - delay?: number         // 出现延迟帧数

行为：
  1. 连线从起点到终点逐步画出（interpolate strokeDashoffset）
  2. 画完后，一个发光圆点沿路径来回流动（循环）
  3. 粒子使用 evolvePath 计算位置
```

### 4.3 SectionTitle

```
输入 props:
  - title: string          // 章节标题
  - subtitle?: string      // 灰色副标题
  - color?: string         // 主题色（默认 #60A5FA）

行为：
  1. 文字从左侧 slide in + fade in（15 帧）
  2. 底部一条 2px 高亮线从左到右展开（spring）
  3. 标题停留，直到场景结束淡出
```

### 4.4 SequenceDiagram

```
输入 props:
  - participants: Array<{
      name: string,
      color: string,
      x: number    // 水平位置
    }>
  - messages: Array<{
      from: number,       // participant index
      to: number,
      label: string,
      delay: number       // 相对延迟帧数
    }>
  - yStart: number         // 第一条消息的 Y 坐标
  - messageSpacing: number // 消息间垂直间距

行为：
  - participant 竖线先出现（从上到下画出）
  - 然后每个消息箭头按 delay 依次从左到右/右到左画出
  - 箭头到达终点时，标签文字 fade in
```

### 4.5 AnimatedBarChart

```
输入 props:
  - data: Array<{ label: string, value: number, color: string }>
  - x: number, y: number
  - width: number, height: number
  - maxValue: number
  - staggerDelay?: number  // 默认 TIMING.barStaggerDelay

行为：
  - 柱子从底部向上 spring 生长
  - 每个 bar 按 staggerDelay 依次出现
  - 标签在柱子上方 fade in
```

### 4.6 CodeBlock

```
输入 props:
  - code: string           // 代码文本
  - language?: string      // 语言标签（显示在右上角）
  - x: number, y: number
  - width: number, height: number

行为：
  - 深色圆角矩形背景（#1E1E1E）
  - 代码逐字 typewriter 出现（每帧 1 个字符）
  - 使用 JetBrains Mono 字体
  - 右上角显示 language 标签
```

### 4.7 RadarScan（远程探测视频专用）

```
输入 props:
  - x: number, y: number   // 中心位置
  - radius: number
  - color?: string          // 默认 #34D399

行为：
  - 3 层同心圆从中心向外展开
  - 一条扫描线从 12 点方向顺时针旋转（3 秒一圈）
  - 扫描线经过时，背景扇区短暂高亮
```
