# 视频 3：运维知识库系统

> **时长**: 40 秒（1200 frames @30fps）
> **目标受众**: 网络运维工程师、AI 应用开发者、技术决策者
> **核心信息**: 运维知识库通过三层对称架构（Agent → MCP → Data），实现多格式文档自动入库 + 语义检索 + Reranker 精排，与带宽管理共享 Agent、隔离 RAG

## Scene 间组装方式

主 Composition（`videos/OpsKnowledge/index.tsx`）使用 `<TransitionSeries>` 串联所有 Scene：

```tsx
import { TransitionSeries, linearTiming } from "@remotion/transitions";
import { fade } from "@remotion/transitions/fade";
import { slide } from "@remotion/transitions/slide";

const SCENES = [
  { Component: Intro,               frames: 90  },
  { Component: ProblemStatement,     frames: 150 },
  { Component: ThreeLayerArch,       frames: 180 },
  { Component: DocumentTypes,        frames: 150 },
  { Component: UploadPipeline,       frames: 210 },
  { Component: SemanticSearch,       frames: 240 },
  { Component: RAGIsolation,         frames: 105 },
  { Component: Outro,                frames: 145 },
];

const TRANSITION = linearTiming({ durationInFrames: 10 });

export const OpsKnowledgeVideo: React.FC = () => {
  return (
    <TransitionSeries>
      {SCENES.map((scene, i) => (
        <React.Fragment key={i}>
          <TransitionSeries.Sequence durationInFrames={scene.frames}>
            <scene.Component />
          </TransitionSeries.Sequence>
          {i < SCENES.length - 1 && (
            <TransitionSeries.Transition
              presentation={i % 2 === 0 ? fade() : slide({ direction: "from-right" })}
              timing={TRANSITION}
            />
          )}
        </React.Fragment>
      ))}
    </TransitionSeries>
  );
};
```

> 注意：7 个转场各 10 帧 = 70 帧重叠。总帧 = sum(scenes) - 70 = 1270 - 70 = 1200 帧 = 40 秒 ✅
