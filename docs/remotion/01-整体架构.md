# 视频 1：DeerFlow 整体架构

> **时长**: 40 秒（1200 frames @30fps）
> **目标受众**: 技术决策者、架构师、AI 应用开发者
> **核心信息**: DeerFlow 是一个 Super Agent Harness，由 4 个容器协作，Agent 通过中间件链 + 工具生态 + 子代理系统完成复杂任务

---

## 分镜表

### Scene 1: Intro（帧 0–90，3 秒）

**内容**: 视频标题淡入

**画面描述**:
- 背景为标准渐变（`#0a0e17` → `#111827`）+ 微弱网格
- 中央显示大标题，分两行：
  - 第一行：`DeerFlow` （JetBrains Mono，72px，白色 `#F9FAFB`）
  - 第二行：`Super Agent Harness` （Noto Sans SC，36px，紫色 `#A78BFA`）
- 标题下方一条细线（`#A78BFA`，2px）从中间向两侧展开

**动画参数**:
```
帧 0–30:   背景渐变 + 网格从 opacity 0 → 0.05
帧 10–50:  "DeerFlow" 文字从 scale(0.5) + opacity(0) → scale(1.0) + opacity(1)，spring(damping:200)
帧 30–60:  "Super Agent Harness" 同上，delay 20 帧
帧 40–80:  底部细线从 width(0) → width(600px)，interpolate 线性

```

## Scene 间组装方式

主 Composition（`videos/Architecture/index.tsx`）使用 `<TransitionSeries>` 串联所有 Scene：

```tsx
import { TransitionSeries, linearTiming } from "@remotion/transitions";
import { fade } from "@remotion/transitions/fade";
import { slide } from "@remotion/transitions/slide";

// 各 scene 的帧数
const SCENES = [
  { Component: Intro,          frames: 90  },
  { Component: FourContainers,  frames: 210 },
  { Component: RequestRouting,  frames: 180 },
  { Component: AgentCore,       frames: 250 },
  { Component: ToolEcosystem,   frames: 170 },
  { Component: ThreeFeatures,   frames: 160 },
  { Component: DataLayer,       frames: 130 },
  { Component: Outro,           frames: 70  },
];

const TRANSITION = linearTiming({ durationInFrames: 10 });

export const ArchitectureVideo: React.FC = () => {
  return (
    <TransitionSeries>
      {SCENES.map((scene, i) => (
        <React.Fragment key={i}>
          <TransitionSeries.Sequence durationInFrames={scene.frames}>
            <scene.Component />
          </TransitionSeries.Sequence>
          {i < SCENES.length - 1 && (
            <TransitionSeries.Transition
              presentation={i % 2 === 0 ? fade() : slide({ direction: "from-right" })}
              timing={TRANSITION}
            />
          )}
        </React.Fragment>
      ))}
    </TransitionSeries>
  );
};
```

> 注意：7 个转场各 10 帧 = 70 帧重叠。总帧 = sum(scenes) - 70 = 1260 - 70 = 1190 帧（约 39.7 秒）。
> 微调最后一个 Outro 到 80 帧，总计精确 1200 帧 = 40 秒。
