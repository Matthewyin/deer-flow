# 视频 1：DeerFlow 整体架构

> **时长**: 60 秒（1800 frames @30fps）
> **目标受众**: 技术决策者、架构师、AI 应用开发者
> **核心信息**: DeerFlow 是一个 Super Agent Harness，由 5 个容器协作，Agent 通过 14 层中间件链 + 7 类工具生态 + 子代理系统完成复杂任务，并集成业务基线监控与数据管理

## Scene 间组装方式

主 Composition（`videos/Architecture/index.tsx`）使用 `<TransitionSeries>` 串联所有 Scene：

```tsx
import { TransitionSeries, linearTiming } from "@remotion/transitions";
import { fade } from "@remotion/transitions/fade";
import { slide } from "@remotion/transitions/slide";

// 各 scene 的帧数
const SCENES = [
  { Component: Intro,          frames: 120  },
  { Component: FiveContainers,  frames: 300 },
  { Component: RequestRouting,  frames: 270 },
  { Component: AgentCore,       frames: 330 },
  { Component: ToolEcosystem,   frames: 270 },
  { Component: FiveFeatures,    frames: 300 },
  { Component: DataLayer,       frames: 210 },
  { Component: Outro,           frames: 120  },
];

const TRANSITION = linearTiming({ durationInFrames: 10 });

export const ArchitectureVideo: React.FC = () => {
  return (
    <TransitionSeries>
      {SCENES.map((scene, i) => (
        <React.Fragment key={i}>
          <TransitionSeries.Sequence durationInFrames={scene.frames}>
            <scene.Component />
          </TransitionSeries.Sequence>
          {i < SCENES.length - 1 && (
            <TransitionSeries.Transition
              presentation={i % 2 === 0 ? fade() : slide({ direction: "from-right" })}
              timing={TRANSITION}
            />
          )}
        </React.Fragment>
      ))}
    </TransitionSeries>
  );
};
```

> 7 个转场各 10 帧 = 70 帧重叠。总帧 = sum(scenes) - 70 = 1920 - 70 = 1850。
> Outro 增加 30 帧至 150，总计精确 1800 帧 = 60 秒 ✅
