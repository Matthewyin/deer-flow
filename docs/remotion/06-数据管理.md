# 视频 6：数据管理系统

> **时长**: 40 秒（1200 frames @30fps）
> **目标受众**: 系统运维、数据管理员、平台使用者
> **核心信息**: DeerFlow Data Manager 是独立 FastAPI 容器，通过 3 个 Tab 功能区统一管理 everybusiness 每日运营数据、应急预案文件上传与向量入库、ECS 远程探测数据的异步采集与解析

## Scene 间组装方式

主 Composition（`videos/DataManager/index.tsx`）使用 `<TransitionSeries>` 串联所有 Scene：

```tsx
import { TransitionSeries, linearTiming } from "@remotion/transitions";
import { fade } from "@remotion/transitions/fade";
import { slide } from "@remotion/transitions/slide";

const SCENES = [
  { Component: Intro,                frames: 90  },
  { Component: ServiceTopology,      frames: 190 },
  { Component: ThreeTabs,            frames: 230 },
  { Component: EveryBusinessEmergency, frames: 210 },
  { Component: ProbeAsyncPipeline,   frames: 240 },
  { Component: ErrorCodeSystem,      frames: 130 },
  { Component: Outro,                frames: 160 },
];

const TRANSITION = linearTiming({ durationInFrames: 10 });

export const DataManagerVideo: React.FC = () => {
  return (
    <TransitionSeries>
      {SCENES.map((scene, i) => (
        <React.Fragment key={i}>
          <TransitionSeries.Sequence durationInFrames={scene.frames}>
            <scene.Component />
          </TransitionSeries.Sequence>
          {i < SCENES.length - 1 && (
            <TransitionSeries.Transition
              presentation={i % 2 === 0 ? fade() : slide({ direction: "from-right" })}
              timing={TRANSITION}
            />
          )}
        </React.Fragment>
      ))}
    </TransitionSeries>
  );
};
```

> 注意：6 个转场各 10 帧 = 60 帧重叠。总帧需精确控制为 1200 帧 = 40 秒 ✅
