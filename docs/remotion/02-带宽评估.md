# 视频 2：带宽评估系统

> **时长**: 40 秒（1200 frames @30fps）
> **目标受众**: 网络运维工程师、SRE、技术架构师

## Scene 间组装方式

主 Composition（`videos/Bandwidth/index.tsx`）使用 `<TransitionSeries>` 串联所有 Scene：

```tsx
import { TransitionSeries, linearTiming } from "@remotion/transitions";
import { fade } from "@remotion/transitions/fade";
import { slide } from "@remotion/transitions/slide";

const SCENES = [
  { Component: Intro,                frames: 90  },
  { Component: ThreeLayerArchitecture, frames: 180 },
  { Component: SingleSourceOfTruth,   frames: 150 },
  { Component: FiveMcpTools,          frames: 180 },
  { Component: RagRetrievalFlow,      frames: 210 },
  { Component: BandwidthTierChart,    frames: 210 },
  { Component: EmailTemplates,        frames: 120 },
  { Component: Outro,                 frames: 120 },
];

const TRANSITION = linearTiming({ durationInFrames: 10 });

export const BandwidthVideo: React.FC = () => {
  return (
    <TransitionSeries>
      {SCENES.map((scene, i) => (
        <React.Fragment key={i}>
          <TransitionSeries.Sequence durationInFrames={scene.frames}>
            <scene.Component />
          </TransitionSeries.Sequence>
          {i < SCENES.length - 1 && (
            <TransitionSeries.Transition
              presentation={i % 2 === 0 ? fade() : slide({ direction: "from-right" })}
              timing={TRANSITION}
            />
          )}
        </React.Fragment>
      ))}
    </TransitionSeries>
  );
};
```

> 注意：7 个转场各 10 帧 = 70 帧重叠。总帧 = sum(scenes) - 70 = 1260 - 70 = 1190 帧（约 39.7 秒）。
> Outro 补齐 10 帧（120 帧），总计精确 1200 帧 = 40 秒 ✅
