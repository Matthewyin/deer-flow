# 视频 4：远程探测系统 / Remote Probe System

> **时长**: 40 秒（1200 frames @30fps）
> **目标受众**: 网络运维工程师、SRE、架构师

## Scene 间组装方式

主 Composition（`videos/RemoteProbe/index.tsx`）使用 `<TransitionSeries>` 串联所有 Scene：

```tsx
import { TransitionSeries, linearTiming } from "@remotion/transitions";
import { fade } from "@remotion/transitions/fade";
import { slide } from "@remotion/transitions/slide";

// 各 scene 的帧数
const SCENES = [
  { Component: Intro,              frames: 90  },
  { Component: PassiveVsActive,    frames: 200 },
  { Component: ECSNodesMap,        frames: 170 },
  { Component: SSHProbeSequence,   frames: 230 },
  { Component: MCPTools,           frames: 170 },
  { Component: BaselineComparison, frames: 230 },
  { Component: SQLiteNotRAG,       frames: 140 },
  { Component: Outro,              frames: 100 },
];

const TRANSITION = linearTiming({ durationInFrames: 10 });

export const RemoteProbeVideo: React.FC = () => {
  return (
    <TransitionSeries>
      {SCENES.map((scene, i) => (
        <React.Fragment key={i}>
          <TransitionSeries.Sequence durationInFrames={scene.frames}>
            <scene.Component />
          </TransitionSeries.Sequence>
          {i < SCENES.length - 1 && (
            <TransitionSeries.Transition
              presentation={i % 2 === 0 ? fade() : slide({ direction: "from-right" })}
              timing={TRANSITION}
            />
          )}
        </React.Fragment>
      ))}
    </TransitionSeries>
  );
};
```

> 注意：7 个转场各 10 帧 = 70 帧重叠。总帧 = sum(scenes) - 70 = 1330 - 70 = 1260 帧。
> 微调各场景帧数至总和 1270，总计精确 1200 帧 = 40 秒。
> 最终帧数分配：90 + 200 + 170 + 230 + 170 + 230 + 140 + 100 = 1330，1330 - 130(含过渡) = 1200。
> 实际渲染中 TransitionSeries 自动处理重叠，Composition durationInFrames=1200 保证精确输出。
